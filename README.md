## 📱 Funcionalidades

| Sección | Descripción |
|---------|-------------|
| **Dashboard** | Métricas clave: total leads, clientes activos, ingresos, tareas pendientes |
| **Clientes** | Lista completa de leads con búsqueda, filtros y edición rápida |
| **Pipeline** | Kanban visual con drag & drop para mover leads entre estados |
| **Tareas** | Gestión de tareas del equipo con asignación y fechas límite |
